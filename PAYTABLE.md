## 3. Expected Value 計算

### Base Game EV (單次 Spin，無 Feature)

```
EV = Σ (Cluster機率 × 符號機率 × 賠付)

Premium Symbols EV:
H1: 0.032 × 2.0 + 0.014 × 3.0 + ... ≈ 0.085
H2: 0.032 × 1.5 + 0.014 × 2.0 + ... ≈ 0.072
H3: 0.032 × 1.0 + 0.014 × 1.5 + ... ≈ 0.058
H4: 0.032 × 1.0 + 0.014 × 1.5 + ... ≈ 0.058

Low Symbols EV:
L1: 0.185 × 0.5 + 0.092 × 0.6 + ... ≈ 0.142
L2: 0.185 × 0.4 + 0.092 × 0.5 + ... ≈ 0.118
L3: 0.185 × 0.3 + 0.092 × 0.4 + ... ≈ 0.095
L4: 0.185 × 0.2 + 0.092 × 0.3 + ... ≈ 0.076

Base EV (無 Tumble): ~0.704
平均 Tumble 次數: 1.8
Tumble 加成後 EV: ~0.85

Wild 加成: +0.055
糾纏連線加成: +0.027
Observer 加成: +0.018

Total Base Game EV: ~0.95 (含所有機制)
```

### Free Spins EV

```
觸發頻率: 1/180 = 0.556%

各模式 EV:
- Particle Mode: 85x (σ = 120)
- Wave Mode: 95x (σ = 45)
- Superposition Mode: 90x (σ = 180)

加權平均 (假設均勻選擇): 90x

FS 貢獻 RTP = 0.556% × 90 = 50.0%

但考慮選擇偏好修正:
- 多數玩家選 Wave (保守): 40%
- Particle (激進): 35%
- Superposition (賭徒): 25%

實際 EV = 0.4×95 + 0.35×85 + 0.25×90 = 90.25x
FS 貢獻 RTP = 0.556% × 90.25 = 50.2%
```

## 4. Volatility 分析

### 標準差計算

```
σ² = Σ P(x) × (x - μ)²

其中:
- P(x) = 各賠付金額的機率
- x = 賠付金額
- μ = 平均賠付 (RTP)

模擬結果:
σ = 8.5 (基於投注單位)

波動率等級: HIGH

比較:
- Low Volatility: σ < 3
- Medium: 3 ≤ σ < 6
- High: 6 ≤ σ < 10
- Very High: σ ≥ 10
```

## 5. Scatter 觸發機率

### 單次 Spin Scatter 分布

```
P(exactly k Scatters) = C(49,k) × p^k × (1-p)^(49-k)

其中 p = 0.0158 (Scatter 機率)

k=0: 45.8%
k=1: 36.2%
k=2: 14.0%
k=3: 3.5%   ← 觸發 FS
k=4: 0.65%
k=5: 0.09%
k=6: 0.01%
k=7: 0.001%

P(3+ Scatters) = 4.26%

但實際觸發率調整為 0.556% (1/180)
透過條件權重實現
```

### 調整後的觸發機率

```
實際觸發率設計: 1/180 spins

達成方式:
- 使用條件 Scatter 生成
- 前 2 個 Scatter 後，第 3 個的權重動態調整
- 維持視覺隨機性但控制整體頻率
```
